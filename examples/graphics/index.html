Graphical examples

Cubes

This example draws some cubes turning around an axe with the same angular speed. Each time the user click on the window, a new cube is created. This example illustrates synchronous composition (all the cubes have the same angular speed), dynamic creation.

Spirals and Lissajous curves

The curves draw by this program are composed of hundreds of small lines. All the lines are processes executed in parallel.
