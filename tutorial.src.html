Tutorial

The tutorial is design to run the ReactiveML interactive mode using rmltop. Lines starting with then # symbol cannot be used in the compiled mode.

Lessons

  • Lesson 1: Quick tour

    An introduction to the main constructs of ReactiveML.

  • Lesson 2: Moving ball

    Illustration of the dynamic creation of processes through a graphical example.

  • Lesson 3: Dynamic reconfiguration

    Examples of the expressiveness of higher order.

  • Lesson 4: Alarm clock

    How to program automata.
