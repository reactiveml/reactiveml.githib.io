Then lets define the clock control signals, 

signal ck_off, ck_bip, ck_snooze;;


signal ck_arm default 0 gather (fun x y -> x);;

Note that signal ck_arm carry a value. This value will be used to set up the countdown of the alarm. On the other hand, when the snooze button is hit, the duration of the countdown is fixed. Hence signal ck_snooze does not need to carry a value.
