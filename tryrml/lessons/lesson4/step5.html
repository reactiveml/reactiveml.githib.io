In this example, time will be only logical to avoid interactions with the operating system. To see how to logical time can be linked to physical time, the interested reader can see the Reactive Asco example.

So, first, we need a process that waits for a given duration d expressed in number of logical steps and at least one. 

let process sleep d =
  for i = 1 to d do
    pause;
  done
  || pause

Using this process, we can define the ring tone, a process that prints Bip Bip Bip Bip periodically on the standard output. 

let process bipbipbipbip =
     loop
       for i = 1 to 4 do
      	 print_string "Bip "; flush stdout;
      	 run sleep 10;
       done;
       run sleep 100;
       print_newline ()
      end
