rmltop : The interactive mode for ReactiveML

Toplevel directives:
  • #run e;;
        spawn the execution of a process e.
  • #exec e;;
        execute a reactive expression e.
  • #suspend;;
        suspend the simulation.
  • #step;;
        execute one instant of the system. Can be used only while the simulation is suspended.
  • #step n;;
        execute n instants of the system. Can be used only while the simulation is suspended.
  • #resume;;
        go back to the sampled mode.
  • #sampling p;;
        change the sampling rate.
  • #quit;;
        exit the toplevel.
  • # e
        execute e in the OCaml toplevel.
